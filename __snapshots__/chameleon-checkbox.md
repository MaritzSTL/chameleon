# `chameleon-checkbox`

#### `shows aria-invalid when invalid`

```html
<input
  aria-describedby="cha-checkbox-error"
  name="cha-checkbox"
  type="checkbox"
  value=""
>
<span class="checkmark">
</span>

```

#### `shows name attribute`

```html
<input
  aria-describedby="cha-checkbox-error"
  name="cha-checkbox"
  type="checkbox"
  value=""
>
<span class="checkmark">
</span>

```

#### `shows required attribute`

```html
<input
  aria-describedby="cha-checkbox-error"
  name="cha-checkbox"
  required=""
  type="checkbox"
  value=""
>
<span class="checkmark">
</span>

```

#### `shows readonly attribute`

```html
<input
  aria-describedby="cha-checkbox-error"
  name="cha-checkbox"
  type="checkbox"
  value=""
>
<span class="checkmark">
</span>

```
